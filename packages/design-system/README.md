# @bossnyumba/design-system

Shared React component library (Tailwind + Radix primitives) used by all four portals. Owns tokens, `Button`, `DataTable`, forms, and the `ScannerCamera` document-capture component.

## Usage

```tsx
import { Button, DataTable } from '@bossnyumba/design-system'

<Button variant="primary" onClick={submit}>Approve</Button>
```
