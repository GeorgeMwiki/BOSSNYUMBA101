# @bossnyumba/design-system

## 0.1.0

### Initial release

- Core primitives: Button, Input, Select, Modal, Table, Card, Badge, Alert, Tabs, Sidebar, DataGrid
- Form primitives: Label, FormField
- Feedback primitives: Alert, Toast, Tooltip, Skeleton, Spinner, Empty, Progress, Pagination
- Layout primitives: Separator, Header, Container
- Data primitives: DataTable, StatCard, EmptyState, Stat
- Overlay primitives: Dialog, Dropdown, DropdownMenu
- Avatar, ErrorBoundary
- ScannerCamera (device-specific)
- Design tokens exported from `@bossnyumba/design-system/lib/tokens`
- Storybook scaffolding with autodocs
